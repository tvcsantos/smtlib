smtlib
======

Generic SMT Front-End For Multiple Solvers

This repository has moved to https://bitbucket.org/tvcsantos/smtlib

Prerequisites
======

This library depends on https://github.com/tvcsantos/scalax and https://github.com/epfl-lara/ScalaZ3. 

You should make sure that you import scalax to your workspace and have ScalaZ3 .jar in the root this project
